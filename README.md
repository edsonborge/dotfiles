# Dotfiles

All my dotfiles.

---

## Preparing workspace

```bash
$ cd ~

$ git clone https://github.com/edsonborge/dotfiles.git

$ cd dotfiles
```

## Dotbot

To install, just run in the root of the package.

```
$ ./install
```

We use [dotbot](https://github.com/anishathalye/dotbot) to manager.
